Load Calculator for Eletronics Product. Build calculaton for customer to calculate actual load and get product suggestion easily.

== Description ==

Now, you can build calculator with unlimited fields for your Energy supply shop to calculate generator or invertor energy and put short-code anywhere of your website. 

Customer can calculate load easily and they will get a suggestion of similer porduct.


Now, let’s see a quick overview of our Load calculator plugin:

https://youtu.be/6x1BzfZd8jg



## Features: ##

**[Build Calculator:]** Build Calculator of your own filed name and field value. 

**[Shortcode:]** Put short-code anywhere, such as post or page.

**[Calculaton:]** Calculate by quentity easily

**[Product Suggestion:]**  Customer can get product suggestion based on customers calculation.

**[Enable/Disable price and Add to cart:]**  User can enable or disable Add to cart button and price.


== Installation ==

= Modern Way: =
1. Go to the WordPress Dashboard "Add New Plugin" section.
2. Search For "AsCode Woo Calculator". 
3. Install, then Activate it.

= Old Way: =
1. Upload `AsCode Woo Calculator` to the `/wp-content/plugins/` directory
2. Activate the plugin through the 'Plugins' menu in WordPress
